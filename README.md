## Listening

 这是一个基于 JAVAFX 与 Python 爬虫的音乐播放器 ，资源来自于网易云 ，界面模仿自网易云 

### 请勿商用或者其它牟利用途

侵权请联系 1411045825@qq.com

#### 功能

- 推荐歌单

- 排行榜

- 收藏歌曲和歌单

- 播放歌曲和MV

- 显示歌词
